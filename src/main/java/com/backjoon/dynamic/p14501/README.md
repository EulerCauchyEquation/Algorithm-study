## 풀이

DP문제이다.  생각보다 점화식은 곧바로 나왔다.  마지막 n일짜에 상담을 한 경우와 아닌 경우.  상담을 한 경우는 단 상담일수가 충족하는 경우일 때만 생각한다. 따라서,

```
sum(n) = values[n] + sum(n-1)  (단, 상담일수 충족)
sum(n) = sum(n-1) 
```

생각보다 문제는 거창해 보였으나, 영양가는 없는 문제 같다.
